¿Qué es una web estática?

Es un sitio que no requiere actualizaciones en contenido. Su objetivo principal es brindar información

Es la más económica, principalmente para empresas que sólo quieren compartir contenido, tales como una web corporativa

Por ejemplo, este sitio es una web estática

WEB ADMINISTRABLES

WEB ADMINISTRABLES

¿Ahora bien que es una web administrable?

Como indica su nombre es un sitio que requiere modificaciones y actualizaciones de contenido.

Por ejemplo, si tienes un sitio de noticias, un sitio con una galería de imagenes, etc.

Si tu web requiere administración de contenido constante, quizás esta sea la opción más apropiada

E-COMMERCE

E-COMMERCE

¿E-COMMERCE?

Un sitio e-commerce o comercio electrónico es uno donde puedes vender un producto u ofrecer servicios.

Puedes agregar métodos de pago en un carrito, gestionar sesiones de usuarios para los clientes, agregar un catalogo de productos, gestión de inventario, etc

Si tu objetivo es vender un producto, quizás esto sea lo ideal.

APLICACIÓN WEB

¿Qué es una aplicación Web?

Es un "programa" diseñado a la medida, es decir tal como lo deseas. Mediante la cual tendrás que acceder por medio de un navegador.

Por ejemplo, un sistema de gestión de biblioteca, de inventario, de restaurantes. En fin, se diseñará un programa vía web de lo que sea necesario.

PS*, también se puede conversar un PWA para el teléfono
